What's the difference between global and block scope?

Ans:Within Global scope , variables that are declared outside of any function are in global scope.It is accessible from any where in the code and In block scope,when variables are called within the "{ curly bracket }" that are block scope and those variables are only accessible inside the block.

How does the event loop works in Javascript?

Ans: Event loop has a task to look at the stack and task queue. When all of the tasks are executed in stack , when stacks remain empty it takes the first task from task queue and pushes on the stack to execute it.
